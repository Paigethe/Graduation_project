## TODO-005 高风险AI对话自动转人工并通知教师（P0）
**目标**：AI高风险触发后自动创建人工会话、提示学生转人工，并通知负责咨询教师。

**子任务**
- 后端
  - 在 `chat/views.py` 的 `ai_chat` 中：命中高风险时创建/获取对应人工会话（基于咨询分配关系）。
  - 生成“转接事件”记录（可用现有 RiskAlert + message 字段标识来源）。
  - 返回响应中带 `handoff_conversation_id` 与 `handoff_required` 标志。
- 前端
  - `ChatPage.vue`：当 `handoff_required` 为 true 时展示转人工提示与按钮（跳转到人工咨询）。
  - 在咨询教师侧“预警/会话列表”中显示AI转接提示（基于 RiskAlert 读取）。
- 文档/验证
  - 更新 `docs/curl-smoke-tests.md` 增加高风险转接脚本。

**交付物**
- API 响应包含转接信息；前端可一键进入人工会话；教师可看到对应预警。
