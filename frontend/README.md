# 前端（Vue3 + Element Plus + Tailwind）

本前端用于「学生心理健康分析与管理系统」毕业设计演示版，已对接后端 REST API（Django + DRF + JWT）。

## 端口
- 前端：`52516`
- 后端：`52517`（Vite 已代理 `/api` 到后端）

## 启动
```bash
cd 02-系统实现代码/frontend
npm install
npm run dev -- --host 0.0.0.0
```

访问：`http://127.0.0.1:52516/`
